# Fullstack Developer

Expert-level fullstack development capability for building complete applications from conception to deployment.

## Core Capabilities

### 1. Frontend Development
- **React/Next.js**: Modern React with hooks, context, routing, SSR/SSG
- **Vue/Nuxt**: Vue 3 composition API, Nuxt for SSR/SSG
- **Angular**: Component architecture, services, routing, forms
- **Vanilla JS**: Modern ES6+, Web APIs, performance optimization
- **CSS/Styling**: Tailwind, SCSS, CSS-in-JS, responsive design
- **State Management**: Redux, Zustand, Pinia, Context API

### 2. Backend Development
- **Node.js**: Express, Fastify, NestJS, serverless functions
- **Python**: FastAPI, Django, Flask, async programming
- **APIs**: RESTful, GraphQL, WebSocket, authentication/authorization
- **Databases**: PostgreSQL, MongoDB, Redis, Prisma, TypeORM
- **Security**: JWT, OAuth, CORS, input validation, rate limiting

### 3. DevOps & Deployment
- **Containerization**: Docker, Docker Compose
- **Cloud Platforms**: Vercel, Netlify, AWS, Google Cloud
- **CI/CD**: GitHub Actions, automated testing, deployment pipelines
- **Monitoring**: Error tracking, performance monitoring, logging

## Project Architecture

Use `scripts/create-project.py` to scaffold projects with proper architecture:

```bash
python scripts/create-project.py --name myapp --stack react-node --database postgres
```

## Development Workflow

1. **Planning**: Requirements → Tech stack → Architecture
2. **Setup**: Project structure → Environment → Database
3. **Development**: Backend APIs → Frontend components → Integration
4. **Testing**: Unit → Integration → E2E tests
5. **Deployment**: CI/CD → Production → Monitoring
